Global AI Memory Test

Open the browser console and navigate to http://localhost:5174

Test Instructions:

  1. Open the console - You should see debug output showing global data
  2. Select different games - Use arrow keys or click to change selection
  3. Check the stats panel - "Times Selected" should increment with each selection
  4. Select the same game repeatedly - AI should escalate sarcasm after 3+ selections
  5. Switch to a different game - AI should comment on the switch
  6. Launch a game - Click Enter or Space to launch, should trigger launch commentary
  7. Reload the page - All stats should persist and AI should comment on previous behavior

Expected Behavior:

  • ✅ Selection counts should increment and persist
  • ✅ AI commentary should change based on selection patterns
  • ✅ Repeated selections should trigger escalating sarcasm
  • ✅ Switching games should trigger relief commentary
  • ✅ All data should survive page reloads
  • ✅ Console debug output should show non-zero values after interactions

Acceptance Criteria:

❌ FAIL if: Any stat shows 0 after interaction

❌ FAIL if: AI commentary doesn't change with selection

❌ FAIL if: Data doesn't persist after reload

✅ PASS if: All stats update, commentary is dynamic, and data persists
