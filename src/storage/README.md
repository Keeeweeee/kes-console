# Storage & Behavior Tracking

This directory contains user behavior monitoring and data persistence.

## Purpose
- User interaction monitoring and event capture
- Gameplay pattern detection and analysis
- Persistent storage across browser sessions
- Data serialization and error handling

## Components
- `BehaviorTracker.ts` - Main behavior monitoring service
- `StorageManager.ts` - Browser storage with fallback mechanisms
- `UserProfile.ts` - User behavior profile management
- `types/` - Storage and tracking type definitions
- `utils/` - Storage utilities and validation
