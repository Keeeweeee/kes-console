# AI Commentary System

This directory contains the AI system for behavior analysis and adaptive gameplay.

## Purpose
- User behavior pattern analysis and detection
- Dynamic difficulty adjustment based on skill level
- Contextual commentary generation with retro personality
- Cross-game learning and personalization

## Components
- `BehaviorAnalyzer.ts` - Pattern detection and skill assessment
- `CommentaryGenerator.ts` - AI commentary with retro personality
- `DifficultyAdapter.ts` - Dynamic game parameter adjustment
- `types/` - AI system type definitions
- `utils/` - AI utility functions and algorithms
